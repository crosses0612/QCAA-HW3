# QCAA HW3 -- Quantum Chemistry

Student ID: B13705049

This repository contains Python implementations and report sources for QCAA Homework 3:

- `problem1.py` -- Minimal-basis H2 FCI block verification.
- `problem2.py` -- Jordan-Wigner, Bravyi-Kitaev, and parity-tapered mappings.
- `problem3.py` -- STO-3G integrals and ansatz circuits.
- `problem4.py` -- VQE and finite-shot estimation.
- `problem5.py` -- QSCI-style sample-based diagonalization for LiH.
- `hw3_common.py` -- Shared integral, mapping, and exact diagonalization utilities.

## Setup

```powershell
conda activate QML_study
pip install -r requirements.txt
```

## Run

```powershell
python problem1.py
python problem2.py
python problem3.py
python problem4.py
python problem5.py
python report/build_report.py
```
